basically there are two ways to access the values written inside the form either use state variables or we can provide the reference of input fields thats where useRef comes into the picture


basically what happens is that useRef return a whole ass object and in that object we have the value of the input boxes so we need to access them through the object

so the stored value will be current.value so for email it will become email.current.value

useRef is used to basically create a reference to that element like an input tag or button
